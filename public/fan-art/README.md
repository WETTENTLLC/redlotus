# Fan Art Directory

This directory contains fan art images for the Red Lotus Fan Art page.

## Adding Images

To add new fan art images:

1. Place image files in this directory
2. Supported formats: JPG, PNG, GIF, WebP
3. Recommended naming: descriptive names like "red-lotus-portrait.jpg"
4. Optimal size: 800x800px or similar square/rectangular dimensions

## Example Images

The Fan Art page will automatically display any images placed in this directory.
Currently, it includes placeholder sample images to demonstrate the gallery functionality.

## Image Guidelines

- Keep file sizes reasonable (under 2MB per image)
- Use descriptive filenames
- Ensure images are appropriate and family-friendly
- Credit the artist when possible (filename can include artist name)

Example: "sarah_k_red_lotus_digital_art.jpg"
